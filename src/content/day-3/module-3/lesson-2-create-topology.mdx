## 🎯 Ne öğreneceksin?

- `CreateTopology` ile topoloji şeması oluşturabileceksin
- `ST_CreateTopoGeo` veya `toTopoGeom` ile geometrileri topolojiye dönüştürebileceksin
- Topoloji tablolarını (`edge_data`, `node`, `face`) sorgulayabileceksin

---

## 📖 Kavram: Topoloji Veri Yapısı

`CreateTopology('isim', srid, precision)` şu tabloları oluşturur:

| Tablo | İçerik |
|---|---|
| `isim.node` | Düğümler (noktalar) |
| `isim.edge_data` | Kenarlar (çizgiler + sol/sağ yüz referansı) |
| `isim.face` | Yüzeyler (poligonlar) |
| `isim.relation` | Feature → topoloji element bağlantısı |

## ▶ Canlı Örnek — Topoloji Oluştur

<RunnableBlock
  sql={`-- postgis_topology extension yükle
CREATE EXTENSION IF NOT EXISTS postgis_topology;

-- Yeni topoloji: 'kadastro_topo', SRID 4326, precision 0.00001 derece
SELECT CreateTopology('kadastro_topo', 4326, 0.00001);`}
  label="Topoloji şeması oluştur"
/>

<RunnableBlock
  sql={`-- Parsel tablosuna topoloji sütunu ekle

  id    SERIAL PRIMARY KEY,

## ▶ Geometri Topolojiye Dönüştür

<RunnableBlock
  sql={`-- Parselleri topolojiye ekle

-- Topoloji düğümlerini gör
SELECT node_id, ST_AsText(geom) AS konum
FROM kadastro_topo.node;`}

## ⚠ Tuzaklar

1. **`CreateTopology` idempotent değil:** Aynı isimle tekrar çağrılırsa hata. `DropTopology('kadastro_topo')` ile önce sil.
2. **Precision çok büyük:** `precision = 1.0` (1 derece) çok geniş snap — yakın kenarlar birleşir. Genellikle `0.00001` (≈1m) güvenli.
3. **PGlite'ta topology desteği:** Extension yüklü olsa bile bazı fonksiyonlar WASM'da çalışmayabilir. Hata alırsan kavramsal anlatım izini takip et.
