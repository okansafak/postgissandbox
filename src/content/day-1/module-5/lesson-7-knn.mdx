## 🎯 Ne öğreneceksin?

- `<->` KNN operatörü ile en yakın N komşuyu verimli bulabileceksin
- `ORDER BY ... <-> ... LIMIT N` kalıbını kullanabileceksin
- `ST_DWithin` + `<->` kombinasyonuyla arama alanını kısıtlayabileceksin

---

## 📖 Kavram: KNN (K-Nearest Neighbor)

`<->` operatörü iki geometri arasındaki **bbox merkezlerine göre yaklaşık mesafeyi** hesaplar — ama asıl özelliği **GIST indeksini ORDER BY'da kullanabilmesidir**.

```sql
-- En yakın 5 nokta — GIST indeks ile çok hızlı
SELECT id, ST_Distance(a.geom::geography, ref.geom::geography)
FROM noktalar a, referans ref
ORDER BY a.geom <-> ref.geom
LIMIT 5;
```

### ST_DWithin ile Birleştir

Büyük tablolarda önce `ST_DWithin` ile coğrafi kutu çiz, sonra KNN sırala:

```sql
WHERE ST_DWithin(a.geom::geography, ref.geom::geography, 100000)  -- 100 km içinde
ORDER BY a.geom <-> ref.geom
LIMIT 5
```

## ▶ Her Nokta İçin En Yakın Komşu

Lateral JOIN ile her şehire en yakın komşuyu bul:

<RunnableBlock
  sql={`WITH sehirler AS (
  SELECT 'İstanbul'  AS ad, ST_SetSRID(ST_MakePoint(28.979, 41.015), 4326) AS geom
  UNION ALL SELECT 'Ankara',   ST_SetSRID(ST_MakePoint(32.854, 39.930), 4326)
  UNION ALL SELECT 'İzmir',    ST_SetSRID(ST_MakePoint(27.143, 38.423), 4326)
  UNION ALL SELECT 'Konya',    ST_SetSRID(ST_MakePoint(32.483, 37.872), 4326)
  UNION ALL SELECT 'Kayseri',  ST_SetSRID(ST_MakePoint(35.485, 38.720), 4326)
)
SELECT
  a.ad AS sehir,
  knn.ad AS en_yakin,
  ROUND(ST_Distance(a.geom::geography, knn.geom::geography)::numeric / 1000, 0) AS km
FROM sehirler a,
LATERAL (
  SELECT b.ad, b.geom
  FROM sehirler b
  WHERE b.ad <> a.ad
  ORDER BY a.geom <-> b.geom
  LIMIT 1
) knn;`}
  label="Her şehrin en yakın komşusu"
/>

## ⚠ Tuzaklar

1. **`<->` bbox mesafesi:** Tam geometri mesafesi değil — küçük geometrilerde fark ihmal edilebilir, büyük poligonlarda farklılık olabilir. Kesin sıralama için `ST_Distance` ile doğrula.
2. **İndekssiz tabloda KNN:** `<->` operatörü GIST indekssiz tablolarda `ST_Distance` kadar yavaş çalışır. `CREATE INDEX USING gist(geom)` şart.
3. **LATERAL JOIN olmadan N-N:** Her satır için en yakın komşu bulmak istiyorsan `LATERAL` kullan — aksi hâlde cross join yaratırsın.
